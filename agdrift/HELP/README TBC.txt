How to create a Help file for AgDRIFT:

- Milt will send "aghelp.doc". Open it in Word and save it in RTF format as "aghelp.rtf"
- Open "Topic Index Map.xls" in Excel and make sure the topic indices are correct.
- Press the "Generate Map Index from this data" button. This will create the file "Topic Index Map.txt", which is an include file for the help compiler
- Open agdrift.hpj in the help compiler (double-click) and click "Save and Compile"
